#### Описание орбиты
<p>Орбита в пространстве однозначно задается пятью параметрами:<p>

* __$a$__ - большая полуось
* __$e$__ - эксцентриситет
* __$\Omega$__ - долгота восходящего узла
* __$\omega$__ - аргумент перигея
* __$i$__ - наклонение
